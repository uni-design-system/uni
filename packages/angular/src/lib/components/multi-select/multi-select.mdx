# Multi Select

The Multi Select component allows users to select multiple options from a predefined list using checkboxes. It provides flexible layout options and integrates seamlessly with forms and filtering interfaces.

## Usage

```typescript
import { UniMultiSelectComponent } from '@uni-design-system/uni-angular';
```

## Methods

- `selectAll()`: Selects all available options programmatically
- `deselectAll()`: Deselects all options programmatically

## Basic Usage

```html
<uni-multi-select
  [options]="availableOptions"
  [selections]="selectedValues"
  (updates)="onSelectionChange($event)"
>
</uni-multi-select>
```

```typescript
export class MyComponent {
  availableOptions = [
    { value: 'option1', label: 'First Option' },
    { value: 'option2', label: 'Second Option' },
    { value: 'option3', label: 'Third Option' },
  ];

  selectedValues: string[] = ['option1'];

  onSelectionChange(selections: string[] | undefined) {
    this.selectedValues = selections || [];
  }
}
```

## Layout Options

```html
<!-- Horizontal layout -->
<uni-multi-select [options]="options" flexDirection="row" checkboxGap="md">
</uni-multi-select>

<!-- Custom spacing -->
<uni-multi-select [options]="options" flexDirection="column" checkboxGap="lg">
</uni-multi-select>
```

## Programmatic Control

```html
<uni-multi-select
  #multiSelect
  [options]="options"
  [selections]="selections"
  (updates)="onUpdate($event)"
>
</uni-multi-select>

<button (click)="multiSelect.selectAll()">Select All</button>
<button (click)="multiSelect.deselectAll()">Clear All</button>
```

## Accessibility

- Uses semantic checkbox inputs with proper label associations
- Supports keyboard navigation (Tab, Space)
- Screen readers announce option labels and selection states
- Follows WCAG accessibility guidelines

## Styling

The component uses the Box layout system for flexible positioning and spacing. Individual checkboxes inherit their styling from the UniCheckboxComponent.
