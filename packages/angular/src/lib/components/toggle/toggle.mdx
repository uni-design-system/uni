# Toggle

The Toggle component is a switch-style form control that allows users to turn options on or off. It implements Angular's FormCheckboxControl interface from the new Form Signals API for first-class Signal Forms integration.

## Usage

```typescript
import { UniToggleComponent } from '@uni-design-system/uni-angular';
```

## Methods

The component implements the FormCheckboxControl interface:

- `markAsTouched()`: Marks the toggle as touched when the user interacts with it

The component also maintains backward compatibility with ControlValueAccessor methods:

- `writeValue(value: boolean)`: Sets the toggle value
- `registerOnChange(fn)`: Registers a callback for value changes
- `registerOnTouched(fn)`: Registers a callback for touch events
- `setDisabledState(isDisabled: boolean)`: Sets the disabled state

## Accessibility

- The component uses semantic HTML with proper `<input type="checkbox">` and `<label>` elements
- Keyboard navigation is supported (Space to toggle)
- Screen readers will announce the label and state correctly
- The toggle is properly associated with its label for accessibility
- Focus indicators are provided for keyboard navigation

## Styling

The toggle uses the theme service to apply consistent colors based on the selected variant. The component includes smooth animations for state transitions and hover effects. The switch design provides clear visual feedback for on/off states with a sliding animation.

## Toggle vs Checkbox

While functionally identical to the checkbox component, the toggle provides a different visual metaphor:

- **Toggle**: Switch-style control that visually represents on/off states
- **Checkbox**: Traditional checkmark control for selecting/deselecting options

Both components implement the same FormCheckboxControl interface and can be used interchangeably in forms.
