# Checkbox

The Checkbox component is a primitive form control that allows users to select or deselect an option. It implements Angular's FormCheckboxControl interface from the new Form Signals API for first-class Signal Forms integration.

## Usage

```typescript
import { UniCheckboxComponent } from '@uni-design-system/uni-angular';
```

## Methods

The component implements the FormCheckboxControl interface:

- `markAsTouched()`: Marks the checkbox as touched when the user interacts with it

The component also maintains backward compatibility with ControlValueAccessor methods:

- `writeValue(value: boolean)`: Sets the checkbox value
- `registerOnChange(fn)`: Registers a callback for value changes
- `registerOnTouched(fn)`: Registers a callback for touch events
- `setDisabledState(isDisabled: boolean)`: Sets the disabled state

## Accessibility

- The component uses semantic HTML with proper `<input type="checkbox">` and `<label>` elements
- Keyboard navigation is supported (Space to toggle)
- Screen readers will announce the label and state correctly
- The checkbox is properly associated with its label for accessibility

## Styling

The checkbox uses the theme service to apply consistent colors based on the selected variant. The component includes smooth animations for state transitions and hover effects.
