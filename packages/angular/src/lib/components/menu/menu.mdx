## Overview

Menus can be linked to any component, but it is recommended to use either the `text-button` or `icon-button` component.

Menus are populated by creating an array of Menu Items (`MenuItem[]`) and binding it to the `Menu` component.
Actions are optional as the `Menu` component also emits the clicked menu item via the `menuItemClicked` output attribute.

### Menu Item Interface

```
MenuItem {
  label: string;
  symbolName?: string;
  action?: () => void;
}
```
