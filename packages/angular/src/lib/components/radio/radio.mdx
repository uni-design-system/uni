# Radio

The Radio component provides radio button functionality for selecting a single option from a group of choices. It implements Angular's FormValueControl&lt;string&gt; interface from the Form Signals API for first-class Signal Forms integration.

## Usage

```typescript
import { UniRadioComponent } from '@uni-design-system/uni-angular';
```

## Methods

The component implements the FormValueControl&lt;string&gt; interface:

- `markAsTouched()`: Marks the radio group as touched when the user interacts with it
- `handleRadioChange(optionValue: string)`: Internal method for handling option selection

## Accessibility

- The component uses semantic HTML with proper `<input type="radio">` and `<label>` elements
- All radio buttons in a group share the same `name` attribute for proper grouping
- Keyboard navigation is supported (Arrow keys to navigate, Space to select)
- Screen readers will announce the label and state correctly
- Each radio option is properly associated with its label for accessibility
- Focus indicators are provided for keyboard navigation
- Disabled options are properly announced to screen readers

## Styling

The radio component uses the theme service to apply consistent colors based on the selected variant. The component includes smooth animations for selection transitions and hover effects. The radio button design provides clear visual feedback for selected and unselected states with scaling animations for the inner circle.

## Usage Examples

### Basic Usage

```html
<uni-radio
  label="Choose your preference"
  name="preference"
  [options]="options"
  [(value)]="selectedValue"
>
</uni-radio>
```

### With Signal Forms

The component implements `FormValueControl<string>`, so it binds directly to a
Signal Forms field with the `[field]` directive:

```typescript
import { Component, signal } from '@angular/core';
import { form, required } from '@angular/forms/signals';
import { UniRadioComponent } from '@uni-design-system/uni-angular';

@Component({
  selector: 'app-preferences',
  imports: [UniRadioComponent /* + Signal Forms directives */],
  templateUrl: './preferences.component.html',
})
export class PreferencesComponent {
  readonly formState = signal({ preference: 'option1' });

  readonly preferencesForm = form(this.formState, {
    preference: [required()],
  });

  readonly options = [
    { label: 'Option 1', value: 'option1' },
    { label: 'Option 2', value: 'option2' },
    { label: 'Option 3', value: 'option3' },
  ];
}
```

```html
<!-- Template -->
<uni-radio
  label="Select an option"
  [options]="options"
  [field]="preferencesForm.preference"
/>
```

### Standalone (two-way binding)
