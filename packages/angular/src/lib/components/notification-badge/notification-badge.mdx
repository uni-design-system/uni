# Notification Badge<ExperimentalBadge />

A notification badge is a visual indicator that signals new, unread, or important content. It wraps any element and displays a small colored indicator positioned in one of four corners.

## Features

- **Content Projection**: Wraps any element using `<ng-content>` - no manual positioning required
- **Three Variants**: Dot, Count, and Pill styles
- **Smart Count Display**: Automatically shows "+" when exceeding maxCount
- **Flexible Positioning**: Four corner positions supported
- **Theme Integration**: Uses design system colors
- **Accessible**: High contrast colors for visibility
- **Angular 21**: Built with modern signals and standalone architecture

## Usage

```typescript
import { UniNotificationBadgeComponent } from '@uni-design-system/uni-angular';

@Component({
  template: `
    <uni-notification-badge
      [count]="messageCount"
      color="warn"
      position="top-right"
    >
      <button>Messages</button>
    </uni-notification-badge>
  `,
  imports: [UniNotificationBadgeComponent],
})
export class MyComponent {
  messageCount = 5;
}
```

## Best Practices

### Placement

- **Top-right corner**: Most common placement for notifications
- **Content projection**: Wrap any element with the badge component using `<ng-content>`
- **Automatic positioning**: No need for manual `position: relative` styling
- **Icon alignment**: Ensure badges don't obscure important icon details

### Color Guidelines

- **Red (danger)**: Urgent notifications, errors, critical updates
- **Blue (primary)**: General notifications, informational updates
- **Green (success)**: Completed actions, positive notifications
- **Orange (warning)**: Warnings, pending actions

### Count Thresholds

- **Use "+" for large numbers**: Default maxCount is 99, showing "99+" for larger values
- **Consider pill variant**: For frequently changing large numbers
- **Dot for binary states**: Use when you only need to show "new" vs "seen"

### Accessibility

- Provide aria-labels for screen readers
- Ensure sufficient color contrast
- Consider motion sensitivity for animated badges

## Terminology

- **Notification Badge**: Standard industry term for the component
- **Count Indicator**: Badge that displays numerical values
- **Pill**: Badge with extended width for larger numbers
- **Dot/Red Dot**: Simple indicator without numbers
- **Status Dot**: Alternative term for dot variant
