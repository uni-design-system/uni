## Overview

The Expand Toggle is a chevron icon button that controls collapsed/expanded state. It pairs
with the [Expand](?path=/docs/components-expand--docs) component: bind the toggle's
`collapsed` model to the Expand's `collapsed` input to build collapsible sections.

The icon rotates 180° between states and shows an "Expand"/"Collapse" tooltip.
