# Multi Select Dropdown

The Multi Select Dropdown component provides a dropdown interface for selecting multiple options from a filterable list. It implements Angular 21 Form Signal patterns with `FormValueControl<T[]>` and combines dropdown functionality with multi-selection capabilities, including search/filter functionality and convenient select/deselect all options.

## Usage

```typescript
import { UniMultiSelectDropdownComponent } from '@uni-design-system/uni-angular';
```

## Methods

- `selectAll()`: Selects all available options programmatically
- `deselectAll()`: Deselects all options programmatically

## Basic Usage

```html
<uni-multi-select-dropdown
  [options]="availableOptions"
  [(value)]="selectedValues"
  placeholder="Select items..."
>
</uni-multi-select-dropdown>
```

```typescript
export class MyComponent {
  availableOptions = [
    { value: 'option1', label: 'First Option' },
    { value: 'option2', label: 'Second Option' },
    { value: 'option3', label: 'Third Option' },
  ];

  selectedValues: string[] = ['option1'];
}
```

## Form Integration

```typescript
import { FormControl } from '@angular/forms';

export class MyFormComponent {
  multiSelectControl = new FormControl<string[]>(['option1']);

  availableOptions = [
    { value: 'option1', label: 'First Option' },
    { value: 'option2', label: 'Second Option' },
    { value: 'option3', label: 'Third Option' },
  ];
}
```

```html
<uni-multi-select-dropdown
  [options]="availableOptions"
  [formControl]="multiSelectControl"
  placeholder="Select items..."
>
</uni-multi-select-dropdown>
```

## Validation and Error States

```html
<uni-multi-select-dropdown
  [options]="options"
  [(value)]="selectedValues"
  [invalid]="hasValidationError"
  [touched]="hasBeenTouched"
  placeholder="Select options..."
>
</uni-multi-select-dropdown>
```

```typescript
export class ValidationComponent {
  hasValidationError = false;
  hasBeenTouched = false;
  selectedValues: string[] = [];

  validateSelection() {
    this.hasValidationError = this.selectedValues.length === 0;
  }
}
```

## Search Functionality

The component includes built-in search functionality that filters options as you type in the dropdown. The search is case-insensitive and matches against option labels.

## Programmatic Control

```html
<uni-multi-select-dropdown
  #multiSelectDropdown
  [options]="options"
  [(value)]="selectedValues"
>
</uni-multi-select-dropdown>

<button (click)="multiSelectDropdown.selectAll()">Select All</button>
<button (click)="multiSelectDropdown.deselectAll()">Clear All</button>
```

## Accessibility

- Uses semantic dropdown and checkbox patterns with proper ARIA labeling
- Supports keyboard navigation (Tab, Space, Enter, Arrow keys)
- Screen readers announce dropdown state, option labels, and selection counts
- Follows WCAG accessibility guidelines for multi-select controls
- Search input is properly labeled for assistive technologies

## Styling

The component integrates with the theme system and uses the dropdown component for consistent styling. The search input and option list inherit styling from the design system's input and checkbox components respectively. The dropdown positioning is handled by the underlying UniDropdownComponent.
