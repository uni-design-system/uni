## Overview

Multi-line text field with the same Signal Forms contract as Input — use it anywhere
free-form text runs past one line (comments, descriptions, feedback). It shares Input's
themed chrome (color, border, error state, focus outline, typeface) and adds
textarea-specific behavior from the theme's `textarea` options: default `rows` and the
CSS `resize` mode. Bind it to a form with the `[field]` directive, or drive `[(value)]`
directly.
