## Overview

Expand is a group of components that can be used in concert to expand/collapse sections of information.
The Expand component is transparent in nature and can nest any element to collapse it out of view.
The Expand Toggle is an optional convenience element that can be used to trigger the expand/collapse animation.
It has been designed to work with the Card Header.
